# Requirements
1. NodeJS v12 Above
2. npm or yarn

# Setup
- Buatlah .env file
- Setelah itu kalian masukkan beberapa text dibawah ini:
```
DISCORD_TOKEN=YOURTOKEN
OSU_TOKEN=YOUROSUTOKEN
PORT=NUMBERPORT
```

- Setelah itu untuk memulai botnya kalian perlu melakukan:
```bash
$ npm install
```
- Bila semuanya sudah, maka tinggal dirun saja:
```bash
$ npm start
```
